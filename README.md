  # Uber Data Breach Analysis

## Description
This project analyzes the Uber data breaches and identifies key cybersecurity vulnerabilities.

## Key Topics Covered
- Data Breach Analysis
- Security Vulnerabilities
- Risk Assessment
- Incident Response

## What I Did
- Studied Uber's 2014 and 2016 data breaches
- Identified causes such as exposed credentials and weak security controls
- Suggested improvements like multi-factor authentication and encryption

## Outcome
Developed understanding of real-world cybersecurity failures and prevention strategies.
